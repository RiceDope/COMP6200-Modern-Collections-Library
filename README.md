## Sequence

- ArrayList:
    - insert
    - append
    - appendAll
    - remove
    - get
    - replace
    - sort
    - sortCopy
    - clear
    - contains
    - firstIndexOf
    - length
- SortedArrayList
    - As above
    - sortOnwards
- Queue FIFO
    - Peek
    - enqueue
    - dequeue
    - size
- Queue LIFO
    - Just a stack
- Priority Queue
    - As above
    - sortOnwards
- Stack
    - Push
    - Pop
    - Peek
